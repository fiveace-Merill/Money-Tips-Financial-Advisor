# Money Tips Financial Advisor  
A web-based AI-powered financial advisor that helps users make smarter financial decisions through budgeting guidance, savings strategies, and real-time investment suggestions.

## Overview
Money Tips Financial Advisor is a modern financial planning assistant built with a mobile-first approach.  
Users interact with an AI chatbot that provides personalized financial recommendations using real-time financial data and Retrieval-Augmented Generation (RAG).

This project is currently under active development.  
Frontend: **Complete (Phase 1)**  
Backend: **In Progress (Phase 2)**  

---

## Key Features
- **AI Chat-based Financial Planning**  
  Users converse with an AI agent to ask questions about budgeting, savings, or investments.

- **Budget Simulation & Projections**  
  The system can model spending, simulate finances, and provide future projections.

- **Real-time Investment Suggestions**  
  Uses external APIs + cached results to deliver data-driven recommendations.

- **RAG-Powered Insights**  
  Contextual financial explanations using vector search and retrieval techniques.

- **Material Design 3 UI**  
  The interface uses a customized M3 design system with a consistent brand color scheme.

---

## Design System
This project uses **Material Design 3** with a customized palette based on the brand color:

**Primary Brand Color:** `#13C9D5`

All color roles (light + dark themes) are defined in `globals.css` using Tailwind v4 CSS variables.
